# KgaboTwitter

A simple Twitter/X-inspired front-end built with HTML, CSS, and
Bootstrap. The application starts with a login page and then redirects
users to the timeline (`home.html`) after a successful login.

## Features

-   Login screen
-   Responsive design
-   Bootstrap Icons
-   Dark mode toggle (CSS-based)
-   Twitter-inspired interface

## Technologies Used

-   HTML5
-   CSS3
-   Bootstrap 5.3
-   Bootstrap Icons

## Getting Started

1.  Clone the repository:

    ``` bash
    git clone https://github.com/your-username/your-repository.git
    ```

2.  Open the project folder.

3.  Open `index.html` in your browser.

## Login

There is **no backend authentication**.

To log in:

-   Enter **any username, email, or phone number**
-   Enter **any password**
-   As long as **both fields are not blank**, the form will submit and
    open `home.html`.

This project is intended for front-end demonstration purposes only.
